# Facts-BA Store

## Development Setup

1. Clone the repository
2. Run

```
yarn install
```

3. Make sure you have Nest CLI Installed

```
npm i -g @nestjs/cli
```

4. Set up database

```
docker-compose up -d
```

## Stack
* MongoDB
* Nest


Populate DB

```
http://localhost:3000/seed
```
